Features & Highlights
1
Add a Property
Click Add Property in the sidebar or top button. Fill in the form — price, address, beds, baths, photos, and description. Everything is optional except price, deposit, and address.
2
Add Photos
Paste image URLs into the photo fields. You can use Unsplash links, your own hosted images, or any publicly accessible image URL. The first image becomes the main gallery photo and listing card thumbnail.
3
Add a Video (YouTube, Vimeo, or MP4)
For YouTube: go to the video → Share → Embed → copy the URL from the src="" attribute (looks like https://www.youtube.com/embed/ABC123).

For Vimeo: same process — Embed → copy the src URL.

For MP4: paste the direct link to the .mp4 file. Instagram videos won't embed directly — download and re-upload to YouTube or a file host.

When a video is set, it replaces the main photo in the desktop gallery and appears as the first slide in the mobile carousel.
4
Google Maps
In the Map Search Query field, type the area name using + instead of spaces. Example: Sugar+Land+Texas or Prosper+Texas. When you have a real address, use it directly: 123+Main+St+Forney+Texas+75126.
5
Properties Save Automatically
All properties you add here are saved in your browser's local storage. They connect to the listings page and property detail page automatically — no code changes needed. Use Export JSON to back up your data or hand it off to a developer to move to a real database later.
6
Edit or Delete Anytime
Go to All Properties, find the listing, and click Edit to update any field, or Delete to remove it. Changes take effect immediately across the site.
Delete Property?
This will permanently remove the listing from your site. This action cannot be undone.
